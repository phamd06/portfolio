## Video Game Saves
Video Game save files are susceptible to being tampered with if not for the usage of cryptography. Most video game save files can be edited using a hex editor, essentially allowing every player to cheat in the game via
changing variables such as money, level and other essential game functions. Typially in modern era games, save files are first encrypted with AES and then signed with HMAC-SHA256.
