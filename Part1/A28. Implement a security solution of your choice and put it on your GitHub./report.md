## Overview
For this activity, I implemented a password strength checker in Python as a simple security solution. The purpose of the program is to help users identify whether their password is weak, medium, or strong based on common password security rules.

## Description
The program checks a password against several basic security requirements, including:
- minimum length
- uppercase letters
- lowercase letters
- numbers
- special characters

Based on how many requirements are met, the password is classified as weak, medium, or strong.

## Why this is a security solution
Weak passwords are a common security problem because they are easier to guess, crack, or reuse across multiple accounts. This tool encourages stronger password creation and helps users understand basic password security practices.

## Implementation
The solution was written in Python using conditional logic and regular expressions. It analyses user input and returns a strength rating.
