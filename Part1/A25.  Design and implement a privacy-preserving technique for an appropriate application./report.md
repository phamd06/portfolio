## Overview
For this activity, I created a short simple program in python that takes input photo files such as .jpeg, .jpg and .png and remove potential metadata that is stored inside of them. In the current day, many photo file formats
contain private information such as geographical information and so this program attempts to essentially copy the same photo but takes only copies the essential photo pixels and not extra (special) data such as the geographic
information. **Note that PIL must be installed and imported.**
