## HTTPS (SSL/TLS Encryption)
Where I observed it:
On websites that use secure connections, identified by the padlock icon in the browser address bar.

Description:
HTTPS is a security protocol that encrypts communication between a user’s browser and a website.

Security purpose:
It helps protect sensitive information such as passwords, personal details, and payment data from being intercepted during transmission.

Limitation:
Although HTTPS secures the connection, it does not guarantee that a website itself is legitimate or safe.

[HTTPS Symbol](https.png)


## Password Manager 
Where I observed it:
In my browser or password manager when logging into online accounts.

Description:
A password manager stores and auto-fills login credentials securely.

Security purpose:
It improves security by helping users create and store strong, unique passwords instead of reusing weak ones across multiple accounts.

Limitation:
If the password manager itself is compromised or poorly secured, multiple accounts could be placed at risk.

## Antivirus / Endpoint Protection
Where I observed it:
On my computer through built-in system protection; Windows Security. 

Description:
Antivirus software scans files, downloads, and system activity to detect and block malicious software.

Security purpose:
It helps protect devices from viruses, malware, spyware, and other harmful programs that could compromise data or system integrity.

Limitation:
Antivirus software may not detect every new or advanced threat, particularly if definitions are outdated or if the attack uses social engineering.

## Firewall
4. Firewall

Where I observed it:
In my computer’s security settings.

Description:
A firewall is a security tool that monitors and controls incoming and outgoing network traffic.

Security purpose:
It helps prevent unauthorised access and can block suspicious or potentially harmful connections between a device and external networks.

Limitation:
A firewall cannot stop every cyber threat, especially if harmful activity is allowed through trusted software or user actions.

Evidence:
[Window's Firewall](Windows_Firewall.png)


## Spam / Phishing Filter

Where I observed it:
In my email inbox.

Description:
A spam or phishing filter is used to identify suspicious, harmful, or unwanted emails before they reach the user’s main inbox.

Security purpose:
It improves security by reducing exposure to phishing emails, malicious links, fake login pages, scams, and dangerous attachments.

Limitation:
Some phishing emails may still bypass filters, while legitimate emails may occasionally be incorrectly flagged as spam.
