## 1. Access Control
My backyard gate is an example of access control because it helps restrict entry to the property and creates a boundary between authorised and unauthorised access. It acts as a first layer of physical security by limiting who can enter the backyard area. \
[Backyard Gate](Backyard_gate.webp)

## 2. Deterrence
The sensor light is an example of deterrence because it automatically activates when motion is detected, increasing visibility and drawing attention to movement around the house. This can discourage intruders from approaching unnoticed, particularly at night. \
[Frontyard Sensor Light](Sensor_Light.webp)

## 3. Authentication
The garage remote is an example of authentication because it allows access only when the correct remote signal is used. It acts as a method of verifying that the person opening the garage has the authorised device. \
[Garage Remote](Garage_Remote.webp)

## 4. Multi-layered Security
A door with two separate locks is an example of layered security because it uses more than one protective measure to secure entry. If one lock is bypassed or fails, the second lock still provides an additional barrier, making unauthorised access more difficult. \
[Door with 2 locks](Double_Lock.webp)

## 5. Monitoring
The full house door alarm system is an example of monitoring and detection because it is designed to identify unauthorised entry and alert occupants when a door is opened unexpectedly. This improves household security by increasing awareness of potential intrusions. \
[Door Alarm System](Door_Alarm_System.webp)
