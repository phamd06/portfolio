# Cybersecurity Portfolio

This repository documents practical cybersecurity activities completed as part of my portfolio development. It includes evidence, reflections, screenshots, notes, and supporting files that demonstrate applied security concepts, privacy awareness, cryptography, vulnerability analysis, and generative AI security testing.

## Repository Structure

- `Part1/` – Activity evidence for practical cybersecurity tasks
- `Part2/` – Reflection, supporting documents, and final portfolio materials

## Part 1 Activities
- A1 – Discover security concepts used on campus
- A2 – Discover security concepts used in public space
- A3 – Discover security concepts used in your house
- A4 – Discover a vulnerable website
- A5 – Discover cryptographic implementation used online
- A6 – Discover cryptographic implementation used offline

- A10 – Discover privacy technique used offline
